## Summary

Backend evidence covers RBAC, 知识库 text/file ingestion/search, RAG answer persistence, 工单 workflow, assignment recommendation, SLA status derivation, admin 统计 APIs, SSE streaming, and Redis-backed queue design. Frontend evidence now covers the public homepage, login/session restore, demo guide, knowledge management with text and file upload, RAG chat with streaming/citations/fallback, ticket list/detail workflow, `智能推荐`, `SLA` list/detail visibility, admin dashboard, and system role management. Final defense preparation is captured by `docs/demo/final-defense-runbook.md`, `docs/demo/v1-live-rehearsal-checklist.md`, and `docs/evaluation/rag-live-evaluation-report.md`, which keep live-provider rehearsal evidence tied to repeatable routes, smoke checks, and RAG metrics.
